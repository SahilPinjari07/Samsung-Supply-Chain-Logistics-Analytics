# 📊 Samsung Supply Chain & Logistics Analytics Dashboard

## 🚀 Overview
This project is an end-to-end **Supply Chain & Logistics Analytics Dashboard** built using **Power BI**. It provides actionable insights into sales, inventory, production, supplier performance, and logistics to enable data-driven business decisions.

---

## 🎯 Objectives
- Analyze supply chain operations and identify bottlenecks  
- Monitor key performance indicators (KPIs)  
- Improve inventory and logistics efficiency  
- Support strategic decision-making with data insights  

---

## 🛠️ Tools & Technologies
- **Power BI Desktop**  
- **DAX (Data Analysis Expressions)**  
- **Excel / CSV Datasets**  
- **Star Schema Data Modeling**

---

## 📌 Key Features
- 📊 Interactive dashboards with slicers and drill-through  
- 📈 30+ DAX measures for KPI tracking  
- 📦 Inventory and stock analysis  
- 🚚 Logistics and shipment performance tracking  
- 🏭 Production efficiency insights  
- 🤝 Supplier performance evaluation  

---

## 📊 KPIs Tracked
- Revenue  
- Profit Margin  
- Inventory Turnover  
- Defect Rate  
- On-Time Delivery %  
- Year-over-Year (YoY) Growth  
- Month-over-Month (MoM) Growth  

---

## 🗂️ Data Model
- Designed using **Star Schema**  
- Integrated **8+ datasets**, including:
  - Sales  
  - Inventory  
  - Suppliers  
  - Production  
  - Logistics  

---

## 🔍 Insights Generated
- Identified key cost drivers and inefficiencies  
- Detected shipment delays and logistics issues  
- Evaluated supplier performance for optimization  
- Improved visibility into inventory health
